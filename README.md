# think-about-ime-titans

Unihertz TitanのIMEを自作するプロジェクト

技術書典で書いた記事(ZliTechBook)に使われたリポジトリです。

## Motivation

Titanのキーボードで使えるキーの数が少なく、また初期搭載のIMEでは追加の記号キーを常時表示できないため、プログラミングやSSHクライアントとしての使用に向いていない。

## 概要

プログラミングやコマンドライン操作によく使われる記号のキーを仮想キーボードとしてTitanの画面上に表示するAndroidアプリケーションです。リポジトリ内のファイル構成は、ANdroid Studioのプロジェクトのそれになっています。

## 動作環境

Android 10.0が動作する[Unihertz Titan](https://www.unihertz.com/ja-jp/products/titan)

他の環境で動作するかの検証は行っておりません。
