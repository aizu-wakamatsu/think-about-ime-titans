# think-about-ime-titans

Unihertz TitanのIMEを自作するプロジェクト

技術書典で書いた記事(ZliTechBook)に使われたリポジトリです。

## Motivation

[Unihertz Titan](https://www.unihertz.com/ja-jp/products/titan)のキーボードは物理キーの数が少なく、また初期搭載のIMEでは物理キーに無いカーソルキーなどを画面に表示できないため、プログラミングやSSHクライアントとしての使用に向いていない。

## 概要

プログラミングやコマンドライン操作によく使われる記号のキーを仮想キーボードとしてTitanの画面上に表示するAndroidアプリケーションです。詳細は`TechBook.pdf`を参照します。

## ファイル構成

`docs`ディレクトリ：記事の下書きが入っています。
`project`ディレクトリ：Android Studioのプロジェクトのそれになっています。
`README.md`：このファイルです。
`TechBook.pdf`：技術書典に向けて書いた記事のPDFファイルです。複数人で編集した書物の、私が担当したセクションの抜粋です。

## 動作環境

Android 10.0が動作するTitan

他の環境で動作するかの検証は行っておりません。
